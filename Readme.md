
# task-model

  Task-model compatible with `scrumcard-view`.

## Installation

  Install with [component(1)](http://component.io):

    $ component install hkjels/task-model

## API

```JavaScript
var task = new Task({
  "id": 1,
  "title": "Move the lawn",
  "assignee": "Henrik Kjelsberg",
  "email": "hkjels@me.com",
  "priority": "medium",
  "estimate": 3
});
```
